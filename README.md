# 📌 Overview

This project is a **frontend clone of the Zomato landing page**, designed to replicate the UI and layout of the popular food discovery platform.

The project focuses on building modern responsive UI using **HTML and CSS**, including structured layouts, card components, and a fully styled footer.

It is created **for learning and demonstration purposes** to practice frontend development concepts.

---

# ✨ Features

### 🏠 Hero Section
- Navigation bar with menu links
- Location and restaurant search bar
- Zomato-style landing section

### 🍴 Services Section
- Order Online
- Dining
- Nightlife & Clubs

### 📚 Collections
- Curated restaurant collections
- Card-based UI with overlay effects

### 📱 App Download Section
- Email or phone link sharing
- Play Store and Apple Store download options

### 📑 Footer Section
- Multiple navigation links
- Social media icons
- Country and language selector
